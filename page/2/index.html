强于昨天的自己，便是进步

Posted on 2018-11-22

​ 未来是什么？我曾无数次问自己，但从来没有得到过答案。提起未来，大多数人觉得遥不可及。那么，现实又是什么样？当前社会很现实，现实是不相信眼泪的，在这个物竞天择，适者生存的世界里，成功都是来自与强者。而我也是一个立志做强者的人。当然，在这路上会遇到很多困扰。我今天要说的就是我们总喜欢和别人做比较，比成绩，比技能，等等…

​ 我们的人生为什么要活在与别人的不断比较里？这一生只有一次，为什么不能活给自己？每个人都有自己的路要走，有自己的选择和价值判断，如果总是和别人做比较，以别人的人生为模板，照着别人的生活方式过着每一天，那我们活着的意义在哪？每个人生来平等，也都有自己的优点和缺点。所以，我们不需要总是和被人作比较，做好自己，强于昨天的自己，便是一种进步。

​ 总与人比较，其实很多时候是认得虚荣心在作怪。人的虚荣心往往源于对自己的不自信。但我们却常常忘了，每个人仅有一次的生命，不是为了跟别人比较才存在在这个世界上的。在这个世界上，我们每一个人都是独一无二的存在，因此我们要活出独一无二的个体，不管阳光明媚或风吹雨打，自己亦是世界上最美的风景。

​ 人生，就像是一座金矿，但“采掘”的结果，总是因为人们勤奋程度不同，得到的回报也不相同。我们要做的，就是要牢牢地抓住稍纵即逝的今天，今天能做的事一定要在今天做，不要把任何计划和希望寄托在未知的明天。今天才是最宝贵的，只有紧紧抓住今天，才能有充实的明天，才能有所作为，有所成就。每天都能充实的度过。强于昨天的自己，便是对自己一天的努力最大的奖励。

​ 人的一生，少不了拼搏的过程，只有拼搏，才能成就美好的人生。我们应该把有限的时间用在提升自己，而不是与别人做比较。做好自己，便是一种成功。

​ 最后，愿你活成自己想成为的模样；愿你在生病时，有人彻夜长伴；愿你在最无趣无力的日子里，仍对世界保持好奇；愿你没有悲伤，每天笑口常开；愿你在精疲力尽时，身边有人可以倚靠。愿你学会释怀，愿你走出半生，归来仍是少年。愿你不在常与人比较，强于昨天的自己，便是进步；

盒子模型

Posted on 2018-08-31

盒子模型由盒子壁（border）、内边距（padding）、盒子内容（width+hight）、外边距（margin）组成。

CSS 框模型概述

元素框的最内部分是实际的内容，直接包围内容的是内边距。内边距呈现了元素的背景。内边距的边缘是边框。边框以外是外边距，外边距默认是透明的，因此不会遮挡其后的任何元素。

提示：背景应用于由内容和内边距、边框组成的区域。

提示：内边距、边框和外边距可以应用于一个元素的所有边，也可以应用于单独的边。

提示：外边距可以是负值，而且在很多情况下都要使用负值的外边距。

CSS伪类和伪元素

Posted on 2018-08-28

CSS 伪类

CSS 伪类用于向某些选择器添加特殊的效果。

属性 描述
:active 向被激活的元素添加样式。
:focus 向拥有键盘输入焦点的元素添加样式。
:hover 当鼠标悬浮在元素上方时，向元素添加样式。
:link 向未被访问的链接添加样式。
:visited 向已被访问的链接添加样式。
:first-child 向元素的第一个子元素添加样式。
:lang 向带有指定 lang 属性的元素添加样式。
锚伪类

在支持 CSS 的浏览器中，链接的不同状态都可以不同的方式显示，这些状态包括：活动状态，已被访问状态，未被访问状态，和鼠标悬停状态。

亲自试一试

提示：在 CSS 定义中，a:hover 必须被置于 a:link 和 a:visited 之后，才是有效的。

提示：在 CSS 定义中，a:active 必须被置于 a:hover 之后，才是有效的。

提示：伪类名称对大小写不敏感。
